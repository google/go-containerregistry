## crane blob

Read a blob from the registry

```
crane blob BLOB [flags]
```

### Examples

```
crane blob ubuntu@sha256:4c1d20cdee96111c8acf1858b62655a37ce81ae48648993542b7ac363ac5c0e5 > blob.tar.gz
```
